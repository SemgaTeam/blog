# Blog
Блог нашей команды

# Blog API MVP
1. CRUD для постов
2. Аутентификация и авторизация членов команды

# Модули
1. main: точка входа в приложение
2. config: вся конфигурация (viper)
3. http: HTTP-сервер (echo), handlers, middleware
4. service: бизнес-логика
5. repository: реализация всех интерфейсов (работа с БД, аутентификация, кэширование, ...)
